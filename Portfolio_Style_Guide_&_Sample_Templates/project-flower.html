Flower Diary

This project was a 2D side scrolling adventure type game with an emphasis on a narrative-story based experience as well as its gameplay. With the development last only 3 weeks, a combined effort of 4 people, Sirawit Yuennan (Project Leader), Nicco Pucci (Co-Programmer), Yitian Zhang (Co-Programmer), and Zhixiu Zhu (Artist), we were able to create an ambitious game in such a small development window.
