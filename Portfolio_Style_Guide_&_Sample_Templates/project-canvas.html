Canvas

This project was an arcade mobile game in a form of an "endless runner" with simple controls and mechanics that can be picked up and played by non-gamers. This game was created using Unity 3D. The 3D art assets were created using Blender, while the 2D assets were created using Clip Studio.

Our first player character prototype model, it was changed due to the team that had another theme in mind.

Our final product of the player character, which is a ship instead of using an animal.

Our game ran based on modular coding that allows us to change the property of the game quickly and effectively.
